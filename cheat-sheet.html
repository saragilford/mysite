Objects 101

Q: What is a JavaScript object?

A: An object is a collection of properties. You can think of it as a way to contain data sets in a single variable. It's similar to a Ruby hash.

Q: How do you create a new object?

A: If you want to create a new JavaScript object, you have a few options. You can create a single object using literal notion, displaying the properties either horizonatally or vertically.

You can also create a single object by using keyword new.

Or you can define an object constructor, and then create objects of the constructed type. This is useful for creating multiple objects with the same properties.

Q: What are the elements of an object?

A: Objects are composed of properties and their values. Properties and values are stored within the object as key-value pairs, with the property as the key and the value as the value.
